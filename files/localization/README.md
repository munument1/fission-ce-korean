# FISSION Language Localization

## Basic info

This folder contains all the customized F.I.S.S.I.O.N. specific interface art files.
All files come in two versions:
    - A 'clean' bitmap version 
    - A 'clean' .psd version

The .psd version contains the clean image, the original image with text (hidden), and a test layer

## Basic usage

1. Open the .psd version
2. Select the 'Text' layer and add new text here - layer will automatically apply Fallout style weathering
3. Turn on the hidden layer to align text with original text positions/meanings
4. When text is complete hide the orignal layer again.
5. Go to 'image/mode/indexed color' and convert the document to the fallout palette (fallout.act included in this folder)
6. Save as a .bmp file in OS/2 format at 8bits
